# University Website – University "Kadri Zeka"

This project is a responsive university website developed using PHP, HTML, CSS, and JavaScript.

The website presents information about the University "Kadri Zeka", including faculties, news, conferences, student services, and contact details.

---

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- PHP
- XAMPP (Apache Server)
- MySQL (optional)
